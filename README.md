# fcamblor's Homebrew tap

Personal Homebrew tap hosting macOS tools maintained by [@fcamblor](https://github.com/fcamblor).

## Usage

```sh
brew tap fcamblor/tap
brew install --cask ai-usages-tracker
```

## Available casks

| Cask | Description |
|------|-------------|
| `ai-usages-tracker` | Menu bar app tracking AI tool usage (Claude, Codex, etc.) |
